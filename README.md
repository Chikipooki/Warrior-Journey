# Управление / Controls
<center>[RUS]</center>

⍇ - Движение персонажа влево.

⍈ - Движение персонажа вправо.

Space -  Прыжок.

<center>[ENG]</center>

⍇ - Moves character to the left.

⍈ - Moves character to the right.

Space -  Jump.
